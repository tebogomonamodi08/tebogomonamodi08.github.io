Arduino Microcontroller(Pool System)

Description:

To interface pool pump readings to an android app, the app was developed with Java on android studio and the data was stored on mySQL. The app had to display the pool temperature, pool pump speed and allow the user to send input to the pump throught the android app. The analogue readings of the pool pump were recorded on an Arduino ino board and then were procced and store as digital readings
